# App Store Screenshots Guide

## Required Screenshot Sizes

### iOS App Store
- **iPhone 6.7"** (iPhone 14 Pro Max): 1290 x 2796 pixels
- **iPhone 6.5"** (iPhone 11 Pro Max): 1242 x 2688 pixels
- **iPad Pro 12.9"**: 2048 x 2732 pixels
- **iPad Pro 11"**: 1668 x 2388 pixels

### Google Play Store
- **Phone**: 1080 x 1920 pixels (minimum)
- **7" Tablet**: 1200 x 1920 pixels
- **10" Tablet**: 1920 x 1200 pixels

## Design Guidelines

### Visual Elements
- Use consistent app branding colors
- Include device frames for professional look
- Add subtle shadows and highlights
- Ensure text is readable at small sizes

### Text Overlays
- Keep captions concise and benefit-focused
- Use high contrast colors for readability
- Position text to not obscure important UI elements
- Consider localization for international markets

### Best Practices
- Show real content, not placeholder text
- Demonstrate core app functionality
- Use consistent lighting and color temperature
- Avoid cluttered or confusing layouts
- Test visibility on different device sizes
